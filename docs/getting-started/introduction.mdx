# emogir.ls Documentation

**emogir.ls** is a customizable link-in-bio platform that lets you create personalized profile pages with:

- **Extensive customization** — colors, fonts, layouts, animations, and effects
- **Interactive widgets** — Discord, VALORANT, GitHub, Last.fm, Weather, TikTok, Chess.com, and more
- **Social integrations** — connect all your social media platforms
- **Rich features** — music players, comments, likes, badges, and analytics
- **Real-time preview** — see your changes instantly in an intuitive dashboard
  All accessible through an intuitive dashboard with real-time preview.

<Admonition type="warning" title="Warning">
  emogir.ls Documentation is **still work in progress** and relies on public
  contribution.
</Admonition>

## Get Started

Create your personalized profile page in minutes. Sign up is **free** and takes just a few seconds.

<Button href="https://emogir.ls/auth/register">Sign Up for Free</Button>

**Quick Start:**

1. **Sign up** — Choose your unique username
2. **Customize** — Personalize your profile in the dashboard
3. **Share** — Your profile is live at `emogir.ls/yourusername`
