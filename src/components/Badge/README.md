# Badge Component

A simple text badge component styled like a button, suitable for displaying dates, labels, or other short text content.

## Usage in MDX

```mdx
import Badge from "@site/src/components/Badge";

<Badge>12/15/2025</Badge>
```

## Props

- `children` (required): The text content to display
- `className` (optional): Additional CSS classes
- All other props are passed through to the underlying element
